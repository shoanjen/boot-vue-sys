推荐路径（MVP）
1 学生：入社 + 报名闭环
登录 → 社团广场 → 社团详情 → 申请加入 → 审核通过 → 活动报名 → 我的报名
2 社团管理员：管理后台
入社审核 → 成员管理 → 活动管理 → 公告发布（含表格与弹窗交互）
3 平台管理员：社团审核
待审核列表 → 审核详情 → 通过/驳回（原因记录）
